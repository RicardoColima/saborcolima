# Sabor Colima

Proyecto de práctica - Clon de Airbnb adaptado para descubrir restaurantes, bares y cafés en el estado de Colima, México.

Plataforma frontend que permite a los usuarios explorar establecimientos gastronómicos, ver sus ubicaciones en un mapa interactivo, guardar favoritos y dejar reseñas. Incluye sistema de autenticación, perfiles de restaurantes con galería de fotos, y panel para que los dueños de negocios registren sus establecimientos.

Desarrollado con Vue 3, Vue Router, Pinia y Leaflet para el mapa.
